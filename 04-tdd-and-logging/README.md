# Module 4 — TDD, Mockito & Logging
**Construct:** Programming Languages

Single Maven project (`tdd-logging-handson/`) covering JUnit 5, Mockito, and SLF4J.

## Mandatory hands-on
**JUnit (1. JUnit_Basic Testing Exercises)**
- [ ] **Exercise 1: Setting Up JUnit** — JUnit 5 on classpath; one passing test proving setup.
- [ ] **Exercise 3: Assertions in JUnit** — assertEquals / assertTrue / assertThrows etc.
- [ ] **Exercise 4: AAA Pattern + Fixtures** — `@BeforeEach`/`@AfterEach`, Arrange-Act-Assert structure.

**Mockito (3. Mockito exercises)**
- [ ] **Exercise 1: Mocking & Stubbing** — `mock()`, `when().thenReturn()`.
- [ ] **Exercise 2: Verifying Interactions** — `verify()`.

**SLF4J (6. SL4J Logging exercises)**
- [ ] **Exercise 1: Logging Error Messages & Warning Levels** — logger at error/warn/info.

## How to run
```bash
cd tdd-logging-handson && mvn test
```
