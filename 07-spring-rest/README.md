# Module 7 — Spring REST using Spring Boot 3
**Construct:** Products & Frameworks · heaviest module

Spring Boot project (`spring-rest-handson/`), extends the Country domain.

## Mandatory hands-on
- [ ] **Create a Spring Web Project using Maven** — Spring Boot web project.
- [ ] **Spring Core – Load Country from Spring Configuration XML**.
- [ ] **Hello World RESTful Web Service** — `@RestController` returning a greeting.
- [ ] **REST – Country Web Service** — endpoint returning Country JSON.
- [ ] **REST – Get country based on country code** — `GET /countries/{code}`.
- [ ] **Create authentication service that returns JWT** (`5. JWT-handson`) — auth endpoint issuing a signed JWT.

## How to run
```bash
cd spring-rest-handson && mvn spring-boot:run
```
